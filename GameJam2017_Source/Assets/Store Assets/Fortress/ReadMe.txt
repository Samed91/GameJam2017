Create your own Fortress in your City with the Fortress Pack.
The models can be used on Mobil or any other platform, it contains a Low Poly version of the structures.
It also has Hand painted High Quality textures.
You can create several different designs from the walls, main structure and several towers, that can stand on the ground or be attached to larger structures, as sown on the Screenshots.
It uses only one Texture, a 2048x2048 TGA

The Pack is grouped in a main folder named "Fortress Pack" and it has a Demo Scene, shown on the Screenshots, and the Prefabs folder, which contains all the elements.
